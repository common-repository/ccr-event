CCR Upcoming Events WordPress Plugin that can help you to add Upcoming Events in your website.

== Description ==

CCR Event WordPress Plugin developed by [CodexCoder](http://www.codexcoder.com/ "CodexCoder").

It has custom Post type option to add your upcoming event. You can easily add these as like a normal post. To use this plugin into any post just use shortcode. 

Shortcode: [ccr-event] 
Shortcode facility to show this plugin into any page or post. 

You can also add this event in sidebar from widget section.

== Installation ==

1. Upload `ccr-event` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[ccr_event]` in your page or post where you want to show FAQs
